Shiner Leadership — Site Files
================================

Structure
---------
index.html          Main HTML file — upload to your web host root
favicon.ico         Multi-size favicon — upload to root directory
css/styles.css      All styles
js/script.js        Page navigation and interactions
images/
  michael-shiner.jpg        Portrait photo
  grain.svg                 Background texture
  sl-monogram.svg           SL monogram mark
  favicon-32.png            32px PNG favicon
  favicon-180.png           180px Apple touch icon
  wordmark-dark.svg         Gold + cream wordmark, transparent bg
  wordmark-black.svg        Black wordmark, transparent bg (print)
  wordmark-white.svg        White wordmark, transparent bg
  wordmark-dark.png         2x PNG, gold + cream
  wordmark-black.png        2x PNG, black — use for email signatures
  wordmark-white.png        2x PNG, white
  wordmark-preview-dark.png   Preview on dark background
  wordmark-preview-white.png  Preview on white background

Deployment
----------
Upload the entire folder contents to your web host root, preserving
the css/, js/, and images/ subdirectories. All paths are relative.

Email signature
---------------
Use wordmark-black.png on a white background, sized to ~220px wide.

Notes
-----
- Fonts load from Google Fonts (internet connection required to render correctly)
- The site uses hash-based routing (#home, #work, etc.) — no server config needed
